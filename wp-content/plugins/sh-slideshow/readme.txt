== Description ==

SH Slideshow is a Wordpress plugin that will create an animated promo slideshow on your site. You can use it to promote any pages, posts, custom post types in your site and pages from outside. It is fully customisable through the admin pages of your blog. Unlike some other slideshow components, SH Slideshow makes use of jQuery JavaScript frameworks and jQuery plugins.

== Installation ==

Installing the SH Slideshow plugin is quick and simple:

1. Begin by downloading the plugin file (ZIP)
1. Extract the files to your local machine
1. Upload the whole sh-slideshow folder (including the folder itself) to your plugins directory - typically http://www.yourdomain.com/wp-content/plugins/
1. Activate the plugin using your Wordpress admin pages
1. Add the slideshow code to your theme

== Frequently Asked Questions ==

= How to add the slideshow to my pages or posts? =

Copy and paste shortcode to the pages or posts.

= Why my slideshow is display all of the images? =

This is because the jQuery file in your theme or other plugins is different version. Please use wordpress default version.
